# Stock Data Analysis and Dashboard using Python

## Project Overview

This project analyzes historical stock prices and company revenue data to identify trends and visualize financial performance. It demonstrates how to extract data from multiple sources, clean it, and present it in a meaningful dashboard using Python.

---

## Objectives

* Extract stock price data using an API
* Collect revenue data using web scraping
* Clean and preprocess raw data
* Visualize stock price and revenue trends
* Build a simple analytical dashboard

---

## Tools and Technologies

* Python
* Pandas
* yfinance
* BeautifulSoup
* Matplotlib
* Requests

---

## Data Sources

* Stock Data: Retrieved using the yfinance library
* Revenue Data: Extracted from HTML tables using web scraping
